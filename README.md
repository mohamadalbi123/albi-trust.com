# Albi Trust Trading Webapp

First foundation build for a trading assessment and journaling product shaped by real trader experience.

## Current flow

- Premium landing page
- 25-question trader assessment
- Result screen with:
  - trader level
  - strongest area
  - main weakness
  - tailored action-plan direction
- Journal area shell with behavior-focused tracking direction

## Current levels

- Level 1: Reactive Trader
- Level 2: Developing Trader
- Level 3: Structured Trader
- Level 4: Advanced Trader

## What makes this product different

- Built from real trading journaling and real behavioral work
- Focuses on discipline, loss acceptance, risk, consistency, and emotional control
- Not designed as a signal tool or indicator-heavy product
- Designed to evolve into:
  - paid tailored action plans
  - login area
  - saved trader profiles
  - daily journaling
  - charts and behavioral insights
